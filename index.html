open·control is a Control Surface for Ableton Live with some unique features.
It adapts to your needs as each element can be configured the way you want.
It has a large and bright display, 6 buttons with visual feedback and 2 sliders.


Is is Open Source, so all the code is available, and it will continue to improve with its user base.




1 controller, 3 flavors

open·control comes in 3 flavors.
Whether you use it on stage or in the studio, with your fingers or your feet, there is a version that will suit you.


{% include feature_row %}

Each version has the same set of features, simply choose how you want to control it.


Control the controls

Each element of the controller can be changed instantly with an easy to use Web Editor.
You can create up to 3 different pages of controls.
No need to reload Live or refresh anything, all changes are applied immediately.



open·control is the only controller available that shows and controls Variations, Scene names, Looper state... along many other unique features.

Specifications

Display:
  • 32x6 pixels White LED Matrix
  • 6 RGB LEDs with adjustable brightness

    • Connectors:
  • USB Type B
  • Mini jack MIDI In, Out
  • Boxed: 8 Jack Inputs (6 Mono for Pedal Switches, 2 Stereo for Expression Pedals)
  • Pocket: 6 Buttons and 2 Knobs

    • Dimensions:
  • Pocket: 15 x 9 X 4 cm
  • Boxed : 16 x 9.5 X 3.5 cm

    • To help make the planet a bit greener, no cables are provided with open.control.
    The USB cable is a standard one (Printer type)


    FAQ


  • Q: Does opencontrol rely on Max For Live ?
    No, there's no need to have Max For Live
  • Q: How many pedals are needed with the Boxed version ?
    With one pedal, you can already do 3 actions, 9 actions with 2 and so forth
  • Q: What version of Live are supported ?
    Any version from Live 10 will work (Lite, Standard, Suite)
